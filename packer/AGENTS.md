# 必要说明

- 测试用的临时文件放到 temp 目录，不要放在根目录
- 需要保留的临时脚本和代码放到 tests 目录，不要放根目录
- 测试用的PE程序样本放在了 temp\samples 目录
- 持久化写入的文档放到 docs 目录，除非用户明确指定
- 可以使用系统已有的任何开发工具，可以自行探索
- 如果用户只是询问，不要自行开始改动代码
- 如果PowerShell不好用，可以用python或node写脚本
